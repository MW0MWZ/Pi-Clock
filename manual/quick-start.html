Quick Start

Get Pi-Clock running in under ten minutes.

What You Need

  • A Raspberry Pi (any model with HDMI — Pi Zero W, Pi 3, Pi 4, Pi 5, etc.)
  • A micro SD card (4 GB minimum)
  • An HDMI display (1080p recommended, 4K supported on Pi 4/5)
  • A power supply appropriate for your Pi model
  • WiFi or Ethernet connectivity (optional for core display, required for live data)

Setup Steps

1

Download & Flash the SD Card

Download the latest Pi-Clock OS image (.img.xz) from the Pi-Clock home page and write it to an SD card using Raspberry Pi Imager. Select "Use custom" in the OS menu and choose the downloaded file.

Need help? Full step-by-step imaging guide with screenshots →

2

Add Your Configuration

After the write completes, Imager ejects the card. Remove it and re-insert it — a drive called "BOOT" will appear.

Use the Config Generator to create a pi-clock-config.txt file with your WiFi, callsign, and location. Copy it to the BOOT drive, then eject safely.

3

Insert, Connect, Power On

Insert the SD card into your Pi, connect the HDMI display, and apply power. Pi-Clock will:

  1. Show the Pi-Clock splash screen immediately
  2. Expand the data partition to fill the SD card (automatic reboot on first boot)
  3. Connect to WiFi (if configured)
  4. Start the world map display
  5. Start the web dashboard

First boot takes around 60 seconds due to the partition resize. Subsequent boots take 20–35 seconds depending on your Pi model.

4

Open the Dashboard

From any device on the same network, open a browser and navigate to:

https://pi-clock.local

If mDNS is not available on your network, use the Pi's IP address instead (shown on the display in the system info applet). The default login credentials are:

  • Username: pi-clock
  • Password: pi-clock

Change your password immediately after first login. You can do this on the System page of the dashboard, or by setting user_password= in pi-clock-config.txt before first boot.

5

Configure Your Station

In the dashboard, set your:

  • Callsign — displayed on the QTH marker
  • QTH location — latitude/longitude (use the "Use My Location" button for automatic detection)
  • Layers — enable the features you want to see (DX spots, satellites, propagation, etc.)

Changes take effect immediately on the display — no reboot required.

What You'll See

After setup, your display will show:

  • A full-screen world map with the day/night terminator (greyline) moving in real-time
  • UTC and local time clocks in the top corners
  • Your home QTH marker with your callsign
  • Any additional layers you enable (DX spots, satellites, grids, etc.)

That's it! Your Pi-Clock is now running. For detailed information about each feature, see the Display Layers and Applets sections. For network-connected features like DX cluster and satellites, make sure your Pi has internet access.
