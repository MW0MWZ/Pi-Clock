Pi-Clock Manual

Complete guide to setting up, configuring, and operating your Pi-Clock world map display.

Pi-Clock is an open-source ham radio world clock that turns a Raspberry Pi and any HDMI display into a real-time map showing day/night, DX cluster spots, satellite tracks, solar weather, HF propagation predictions, and more. This manual covers everything from first boot to advanced configuration.

New to Pi-Clock? Start with the Quick Start guide to get up and running in minutes, then explore the other sections at your own pace.
