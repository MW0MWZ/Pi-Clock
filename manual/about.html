About Pi-Clock

An open-source ham radio world clock for Raspberry Pi.

What is Pi-Clock?

Pi-Clock is a fully open-source (GPL-2.0) ham radio world clock, inspired by the classic GeoChron concept but built from scratch as an independent project. It runs on a Raspberry Pi connected to any HDMI display and shows a high-resolution world map with real-time astronomical, propagation, and DX data.

The display composites NASA satellite imagery (Blue Marble for daytime, Black Marble city lights for night) with a physically accurate greyline terminator, then overlays live ham radio data: DX cluster spots, satellite ground tracks, HF propagation predictions, solar weather indices, and a scrolling news ticker.

Everything is configurable through a web-based dashboard accessible from any browser on your local network. Layers can be toggled on and off, opacity adjusted, and features configured without touching the Pi itself.

Key Features

Real-Time World Map

NASA Blue Marble (day) and Black Marble (night) imagery with a smooth greyline terminator that updates every 60 seconds. Beautiful at 1080p or 4K.

DX Cluster Integration

Live DX spots plotted on the map with great-circle arcs, band-coloured markers, and a scrolling feed panel. Connects to any DX Spider or AR Cluster server via telnet.

Satellite Tracking

SGP4 orbit propagation for ISS, QO-100, and 15+ amateur satellites. Ground tracks, visibility footprints, and real-time position markers with automatic TLE updates from Celestrak.

Solar Weather & Propagation

Live NOAA SWPC data (SFI, Kp, solar wind, X-ray flux, SDO images), MUF estimates for five directions, and a full VOACAP-style band/hour/distance heat map.

News Ticker

Scrolling headlines from ARRL, RSGB, DX World, Southgate ARC, and NOAA space weather alerts. Smooth scroll on multi-core Pi, headline mode on single-core.

Lightning & Earthquakes

Real-time lightning strikes from Blitzortung.org with attract-flash effect and colour fade. USGS earthquake data (M4.5+) with seismograph icons and concentric ripple rings.

Weather Overlays

Global wind streamlines, rain radar with NWS colour scheme, cloud cover, and aurora oval from NOAA GFS and SWPC. All bilinear-smoothed with surface caching.

Web Dashboard

Configure everything from your browser over HTTPS. Layer controls, DX filters, satellite selection, WiFi, hostname, password, OS upgrades, and system management.

Design Philosophy

  • Small, fast, and light — every CPU cycle matters on a single-core Pi Zero W. The renderer is written in C and layers are only re-rendered when their data changes.
  • Minimal dependencies — only SDL2, Cairo, and Mesa. The dashboard is a single static Go binary. No X11, no Wayland, no desktop environment.
  • Offline-first — the core display (map, greyline, clock, grids, QTH marker) works with no network. Live features (DX, satellites, solar, news) require connectivity but degrade gracefully.
  • GPU over CPU — compositing is offloaded to the VideoCore IV GPU where possible, keeping the CPU free for data processing.
  • Open and extensible — all source code is GPL-2.0 and written to teaching quality with thorough comments. Community contributions welcome.

How It Compares to HamClock

The primary alternative for a ham radio world clock display is HamClock by WB0OEW. Pi-Clock was created to address several limitations:

Feature Pi-Clock HamClock
Source code Open source (GPL-2.0) Closed source
Native resolution 1080p / 4K 800x480 base (scales up)
Base maps NASA Blue & Black Marble satellite imagery Stylised vector maps
Day/night blend Smooth photorealistic terminator Hard boundary
Configuration Web dashboard (any browser) Touch/mouse on the display itself
Layer system 15 independent layers with per-layer opacity Fixed layout
Propagation model VOACAP band/hour/distance heat map + MUF VOACAP band conditions
DX cluster Multi-cluster, band colours, great-circle arcs Single cluster
OS upgrades A/B slot with automatic rollback Manual update
Development Community (open contributions) Single developer

Note: HamClock is an excellent piece of software and WB0OEW deserves great credit for building and maintaining it. Pi-Clock simply takes a different approach — open source, native high resolution, and community-driven development.

Project Components

Pi-Clock is delivered as three Alpine Linux packages:

Package Language Purpose
pi-clock-renderer C Full-screen map display with all layers. Renders directly to the framebuffer via SDL2 — no desktop environment needed.
pi-clock-dashboard Go Web-based management UI. Single static binary with embedded HTML/CSS/JS. HTTPS with auto-generated certificate.
pi-clock-maps Data NASA satellite imagery at four resolutions (720p, 1080p, 1440p, 4K), country borders, timezone boundaries, CQ zones, and the callsign prefix database.
